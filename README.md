# Dislexicare

Aplikasi mobile berbasis Artificial Intelligence untuk mendeteksi gangguan disleksia pada siswa sekolah dasar menggunakan algoritma GRU.

## Teknologi
- Flutter
- FastAPI
- Python
- MySQL
- PHP

## Fitur Utama
- Deteksi disleksia berbasis teks
- Analisis audio membaca
- Riwayat hasil deteksi
- Profil pengguna

## Dataset
100 data teks dan audio.
